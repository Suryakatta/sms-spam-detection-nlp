# SMS Spam Detection using NLP

## Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify SMS messages as Spam or Ham (Not Spam).

## Dataset
- Total Messages: 5572
- Categories:
  - Ham (Legitimate SMS)
  - Spam (Unwanted SMS)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Workflow
1. Load Dataset
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. TF-IDF Vectorization
6. Model Training
7. Prediction
8. Evaluation

## Model Performance
- Training Accuracy: 99.37%
- Testing Accuracy: 98.56%
- Weighted F1 Score: 0.99

## Results
The model successfully classifies SMS messages with high accuracy and minimal overfitting.
